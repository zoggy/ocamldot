OCamldot

What is OCamldot ?

OCamldot is a small library to parse, print and display graphviz dot files.

OCamldot was previously part of Cameleon but is now developped separately and is findlib compatible.

Download

OCamldot is hosted on Github.

Releases

  • 1.0 [2012-04-11]: first release after extraction from Cameleon.

Installation

OCamldot requires LablGtk >= 2.14.1, with gnomecanvas support enabled, installed with ocamlfind to provide the gtk interface.

To compile and install:

  • tar xvfz ocamldot-XXX.tar.gz
  • cd ocamldot-XXX
  • make all install
